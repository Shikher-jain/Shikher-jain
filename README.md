## Projects

### 🤖 [Sahayak AI — Multimodal RAG Assistant](https://github.com/Shikher-jain)
*Python · FastAPI · PyTorch · HuggingFace · Qdrant · FAISS · Docker · Streamlit*

A multimodal RAG system for processing PDFs, images, audio, and video through semantic retrieval. Built fault-tolerant hybrid storage (Qdrant + FAISS/SQLite fallback) with modular FastAPI microservices. Achieved **sub-200ms latency** under concurrent workloads via HNSW indexing, caching, and async request handling. Deployed via Docker.

---

### 🔍 [Pathwise AI — Job Outreach Engine](https://github.com/Shikher-jain)
*Python · FastAPI · PostgreSQL · Sentence-Transformers · Streamlit*

NLP-driven job-resume matching using embedding-based semantic similarity with TF-IDF fallback. End-to-end pipelines for multi-source job aggregation, resume parsing, skill extraction, and automated personalized cold email generation. Improved matching relevance by **30%** and reduced manual effort by **50–60%**.

---

### 📊 [Data Vista — Unified Analytics & ML Platform](https://github.com/Shikher-jain)
*Python · FastAPI · Flask · Scikit-learn · Plotly · PostgreSQL · Docker*

A modular service-oriented platform integrating 12+ backend services with unified REST APIs for analytics, ML inference, and data processing. Covers EDA, feature engineering, regression and tree-based models with cross-validation, and interactive Plotly/Streamlit dashboards. Deployed production-ready on Render.
